# youyoungshin.com

Single-file portfolio site. Everything lives in `index.html`; images and videos in `/images`.

## Editing

Open `index.html` in any editor. All content lives in JavaScript arrays near the top of the `<script>` block:

- `WORKS` — every artwork (title, year, category, materials, statement, images)
- `COLLAGE` — home page layout
- `EXHIBITION_ITEMS`, `CV`, `SHOUTOUT`, `SHOUTOUT_ARTISTS`
- `ABOUT_TEXT`, `LINKS`

To add a new image: drop it into `/images` and reference it as `images/filename.jpg`.

## Deploying

Any static host works. This site is deployed on GitHub Pages.
